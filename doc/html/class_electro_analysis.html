Detailed Description

Class to obtain the macroscopic electrical quantities.

The main simulation computes the time evolution of the mean field quantities of number density and velocity. From these the appropriate macroscopic quantities need to be derived in order to benchmark the numerical results with literature or experiments. This class provides the methods to compute such quantities from the data stored on the GrapheneFluid1D or GrapheneFluid2D classes.

float ElectroAnalysis::AverageDirectCurrent ( const GrapheneFluid2D &  graphene)
static

2D Integrator for the average drain-to-source current

The average drain-to-source current \( I_{DS}=\frac{1}{L}\int_0^L I(x)\,dx \) with \( I(x) = \int_0^W ej_x \,dy \). Thus

\[ I_{DS}= en_0v_0L \int_0^{w}\int_0 j_x^*\,dx^*dy^* \]

and the current prefactor is given by \( en_0v_0L\) This method computes the integral \( \int_0^{w}\int_0^1 j_x \,dxdy \)

◆ OhmPower() [2/2]

float ElectroAnalysis::OhmPower ( const GrapheneFluid2D &  graphene)
static

2D Integrator for the dissipated Ohm power.

The dissipated Ohm power can be estimated with \( P_\Omega = J\cdot E \) nothing that \( J = \sigma E \) and \( \sigma=ne^2\tau/m^\star \doteq \sigma_0\sqrt{n^*} \). Therefore,

\[ P_\Omega = \int_0^W\int_0^L \frac{j_x^2+j_y^2}{\sigma}\,dxdy = \frac{e^2v_0^2n_0^2L^2}{\sigma_0} \int_0^w\int_0^1 \frac{{j_x^*}^2+{j_y^*}^2}{\sqrt{n^*}}\,dx^*dy^* \]

This method computes the integral \( \int_0^{w}\int_0^1 |j|^2/\sqrt{n} \,dxdy \)
