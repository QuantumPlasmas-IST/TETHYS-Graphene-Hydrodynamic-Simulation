Smooth staircase function.

This function is defined as

\[ f(x) = \frac{\tanh a(-\frac{x-1}{w} - \lfloor-\frac{x-1}{w})\rfloor - 1/2)}{2\tanh a/2 } + 1/2 +\lfloor-\frac{x-1}{w}\rfloor \]

corresponding to a smooth descending staircase.

Parameters

step_widthwidth of each step, \( w \)
smoothnesscontrols the smoothness of the jump, \( a \)
