Detailed Description

Graphene electronic fluid class in one dimension.

The GrapheneFluid1D class describes a fluid governed by the usual continuity and Cauchy momemtum equations, where the mass of the fluid element is not constant. It overrides class Fluid1D necessary methods in order to describe the semi-classical electronic fluid.
